# 🎫 E-Ticketing System

A modern web-based E-Ticketing System that allows users to search, book, manage, and cancel tickets for buses, trains, and events. The application provides an interactive seat selection system, booking management, user authentication, and an admin dashboard for managing routes, events, and bookings.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login
- Profile Management
- Search Bus, Train & Event Tickets
- Interactive Seat Selection
- Real-Time Fare Calculation
- Ticket Booking & Cancellation
- Download Ticket Details
- Booking History Management
- Responsive User Dashboard

### 🛠️ Admin Features
- Admin Authentication
- Add New Routes & Events
- Update Existing Listings
- Delete Routes & Events
- View Customer Bookings
- Dashboard Analytics
- Revenue Monitoring
