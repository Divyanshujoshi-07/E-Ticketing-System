# E-Ticketing System - Main Features

## User Module

- User Registration
- User Login
- Profile Management

## Ticket Module

- Search Routes/Events
- Sort Routes/Events by date, price, seats, or name
- Bus, Train, and Event slider to view one category at a time
- View Available Seats
- Seat legend and selected fare calculation
- Book Ticket
- Cancel Ticket
- Download Ticket
- Filter My Bookings by status

## Admin Module

- Add Routes/Events
- Update Routes
- Delete Routes
- View Bookings
- Dashboard summary for listings, booked seats, bookings, and revenue

## Demo Notes

- Open `index.html` in a browser.
- Admin code: `admin123`
- Data is saved in the browser using `localStorage`.
- Sample data includes 10 bus routes, 10 train routes, and 10 events.
